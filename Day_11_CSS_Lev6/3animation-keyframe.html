Keyframes:-


        Example:-  /* Method - 1 to using keyframe  */

         @keyframes nitesh {

            from{
                width: 150px;
                background-color:aqua;
            }
            
            to{
                width: 100%;
                background-color: crimson;
            }
    
        }
    
                /* Method - 2 to using keyframe  */
    
                @keyframes nitesh {
                    
                    33%{
                        width: 33%;
                        background-color:orange;
                    }
                    
                    66%{
                        width: 66%;
                        background-color: white;
                    }
    
                    100%{
                        width: 100%;
                        background-color: green;
                    }
                }
