Grid-template-area:- Using this property customization can apply.

        Example:-  .item {

             display: grid;
             grid-template-columns: 200px 400px;
             grid-template-rows: 30px 500px 30px;
    
        grid-template-areas: 
        "hd hd"
        "side main"
        "ft ft"; }
    
         #header{
            grid-area: hd;
        }
        #sidebar{
          grid-area: side;
         }
         #content{
          grid-area: main;
         }
        #footer{
          grid-area: ft;
        }
