Use of CSS Transform :-
  • translate() :- Used in movement
  • rotate() :- Used in rotation
  • scale() :- To zoom in
  • skew() :- For tilt
  • mattrix() :- all these above property in form of mattrix
Example =>
  1. transform: translate(100px,50px);
  2. transform: rotate(45deg);
  3. transform: scale(2,10);
  4. transform: skew(45deg);
  5. transform: matrix(1.2, 0.2, -1, 0.9, 0, 20);
